# 零基础入门学习Python
<h3>课程概览</h3>
<h4>课程介绍</h4>
本系列教程面向零基础的同学，是一个深入浅出，通俗易懂的Python3视频教程。
前半部分主要讲解Python3的语法特性，后半部分着重讲解Python3在爬虫、Tkinter、Pygame游戏开发等实例上的应用。整个系列共16个章节，前边13个章节从一个小游戏引入Python，逐步介绍Python的语法以及语言特色。最后3个章节为案例的演示，是前边内容的总结和提高。
